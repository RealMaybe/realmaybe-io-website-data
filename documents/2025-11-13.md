## 5. 代码块

### 行内代码

使用 `printf()` 函数输出内容。

### 代码块（无语法高亮）

```
function example() {
  console.log("这是一个代码块示例");
}
```

### 带语法高亮的代码块

```javascript
function greet(name) {
    return `Hello, ${name}!`;
}

console.log(greet("World"));
```

```python
def fibonacci(n):
    if n <= 1:
        return n
    else:
        return fibonacci(n-1) + fibonacci(n-2)
```
